Retrieves the next token from the input file. 


\begin{DoxyParams}{Parameters}
{\em file} & The file to read from. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}

This function implements a finite state machine (FSM) to parse the input file character by character, constructing tokens based on the defined states. It handles various token types, including identifiers, keywords, literals, operators, and punctuation.
